## How to Add a Feature

1. Create folder: `feature-name/`
2. Add files:
   - `01_requirements.md`
   - `02_design.md`
   - `03_api_spec.md`
   - `04_data_model.md`
   - `05_decisions.md`
   - `discussion/` (folder for decision docs)
3. Link in this index
4. Open GitHub Issue for tracking

See [CONTRIBUTING.md](../.github/CONTRIBUTING.md) for details.
