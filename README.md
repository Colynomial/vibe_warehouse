# Vibe Warehouse

A self-service data platform for SMEs to understand their data through AI-powered dashboards and analytics.

## What is Vibe Warehouse?

Vibe Warehouse allows non-technical users to:
- Connect data from multiple sources (Salesforce, Stripe, Google Ads, etc.)
- Build custom dashboards using AI ("vibe coding")
- Create data marts and insights ad-hoc
- Share data safely across teams
